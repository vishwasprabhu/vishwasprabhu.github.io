A Scalable Song Recommendation System GitHub

● Developed a novel pipeline to convert the One Million Song dataset from HDF5 to parquet format. Performed EDA of the artist-level features to study the attributes that contribute to song popularity.

● Built a scalable memory-based and lyrics-based song recommender system, song popularity, and genre predictor by processing Million Songs dataset (1 Tb) and EchoNest users dataset (98 Gb) with distributed computing tools such as Pyspark, MongoDB, and Databricks.

● State-of-the-art distributed models for collaborative filtering, K-means clustering, multi-class classification, and regression tasks were developed and fine-tuned using Spark MLlib. A UI web app was also developed using Flask and HTML to visualize song recommendations.
